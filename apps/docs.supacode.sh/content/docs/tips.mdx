## Naming worktrees

Supacode creates worktrees with random branch names by default. There are a few ways to give them meaningful names.

### Rename the branch directly

```bash
git branch -m "feature/new-login"
```

### Tap the branch name in the top bar

Click the branch name at the top of the terminal to open the rename dialog.

![Rename Branch](/rename-branch.png)

### Let your agent do it automatically

Add this to your `AGENTS.md` (or `CLAUDE.md`, `.cursorrules`, etc.):

```markdown
Before you go on your task, check the current git branch name,
if it's something generic like an animal name, name it accordingly.
Do not do this for main branch.
```

The agent will rename the branch to match the task before starting work.
