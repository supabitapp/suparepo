## Requirements

Supacode requires **macOS 26 Tahoe** or later. The Liquid Glass design language makes it difficult to provide good UI/UX on older OS versions.

## Installation

1. Download Supacode from [supacode.sh](https://supacode.sh)
2. Move the app to your Applications folder
3. Open Supacode

## First Run

When you first open Supacode, you'll see a terminal window. You can:

1. Navigate to your project directory
2. Start your preferred coding agent (Claude Code, Codex, Opencode, etc.)
3. Create worktrees for parallel tasks

## Setting Up Agents

Supacode follows the **BYOA (Bring Your Own Agent)** philosophy. Install your preferred coding agents:

```bash
# Claude Code
npm install -g @anthropic-ai/claude-code

# Or any other agent you prefer
```

Then simply run them in Supacode's terminal — no translation layer needed.

## Auto-Starting Agents

You can configure Supacode to automatically start a coding agent when opening a repository:

1. Open Settings with **⌘,**
2. Add a setup script for your repo
3. The agent will start automatically when you open that repo
