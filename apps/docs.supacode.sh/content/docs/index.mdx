# Supacode

**Run 50+ coding agents in parallel.** Blazing fast native macOS app.

Supacode is a fully native macOS application built with libghostty as the terminal engine. It's designed for developers who want to run multiple AI coding agents simultaneously without compromising on performance.

## Key Features

- **Fully native app** — Native macOS app with libghostty as the engine
- **BYOA — Bring your own agents** — Run Claude Code, Codex, Opencode in terminal without any translation layer
- **Worktree integration** — Isolated git worktree per task. Give each agent a free space to do its thing
- **GitHub integration** — Open PRs, see CI checks, fix conflicts, and more

## Requirements

- macOS 26 Tahoe or later
