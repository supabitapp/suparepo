## Adding repository

Add a git repository to the app.

![Add Repository](/add-repository.png)

After adding a new repo, you should see your repo and its worktrees (if any) in the sidebar.

![Worktree Sidebar](/worktree-sidebar.png)

- Create a new worktree with ⌘ + N

## Auto-Starting Coding Agents

You can configure Supacode to automatically start a coding agent when creating a new worktree:

1. Open Settings with **⌘,**
2. Add a setup script for your repo

For instance:

```bash
claude --dangerously-skip-permissions
```

![Setup Script](/setup-script-claude.png)
