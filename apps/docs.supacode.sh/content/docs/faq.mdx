# Frequently Asked Questions

## Why macOS 26 Tahoe only?

Liquid Glass makes it difficult to provide good enough UI/UX on older OS versions, hence the restriction.

## How do I get proper naming for my worktrees?

Ask your agent to do it before a task. This keeps your parallel workstreams organized and easy to identify.

## How do I configure the Terminal?

Supacode respects your Ghostty config. Do your configurations there at `~/.config/ghostty/config`.

## Where is my Git client?

Opening lazygit as a split works out really well. Install it with `brew install lazygit` and run it alongside your agent.

## How do I start a coding agent automatically?

Start it as a setup script for the repo in Settings (**⌘,**). The agent will launch automatically when you open that repository.

## What agents are supported?

Any terminal-based coding agent works with Supacode:

- Claude Code
- Codex
- Opencode
- Aider
- And more...

Supacode runs agents directly in the terminal without any translation layer.
