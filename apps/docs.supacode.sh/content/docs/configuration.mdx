# Configuration

## Terminal Configuration

Supacode uses libghostty as its terminal engine and **respects your Ghostty configuration**. To customize the terminal:

1. Edit your Ghostty config file (`~/.config/ghostty/config`)
2. Changes will apply to Supacode

See the [Ghostty documentation](https://ghostty.org/docs) for all available options.

## Git Client

Supacode doesn't include a built-in Git client. Instead, we recommend using **lazygit** as a split terminal:

```bash
# Install lazygit
brew install lazygit

# Run it in a split
lazygit
```

This approach gives you a full-featured Git UI while keeping Supacode focused on agent management.

## Worktree Naming

For better organization, ask your coding agent to name worktrees appropriately before starting a task. This helps keep your parallel workstreams organized.
