# Getting Started

## Installation

1. Download Supacode from [supacode.sh](https://supacode.sh)
2. Open the `.dmg` file and drag Supacode to your Applications folder
3. Launch Supacode

## Terminal Configuration

Supacode respects your Ghostty config. Configure your terminal preferences there.

## Starting a Coding Agent

You can start coding agents like Claude Code, Codex, or Opencode directly in the terminal.

To start an agent automatically when opening a repo, configure it as a setup script in Settings (⌘,).

## Git Worktrees

Each task gets an isolated git worktree, giving each agent a dedicated space to work without conflicts.

## FAQ

### Why macOS 26 Tahoe only?

Liquid Glass makes it difficult to provide good enough UI/UX on older OS versions, hence the restriction.

### How do I get proper naming for my worktrees?

Ask your agent to do it before a task.

### Where is my Git client?

Opening lazygit as a split works out really well.
